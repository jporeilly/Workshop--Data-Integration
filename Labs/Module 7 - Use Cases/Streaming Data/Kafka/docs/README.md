# Workshop Documentation

## Hands-On Workshop Labs

Self-contained, step-by-step labs — each can be completed independently:

| # | Workshop | Difficulty | Duration |
|---|----------|-----------|----------|
| 1 | [Real-time User Activity Stream](scenarios/workshop-1-user-activity.md) | Beginner | 30–45 min |
| 2 | [High-Frequency Stock Trades](scenarios/workshop-2-stock-trades.md) | Intermediate | 45–60 min |
| 3 | [E-Commerce Purchases — Avro with Schema Registry](scenarios/workshop-3-purchases-avro.md) | Intermediate | 45–60 min |
| 4 | [Time-Bounded Data Retrieval](scenarios/workshop-4-time-bounded.md) | Intermediate | 45–60 min |
| 5 | [Multi-Topic Consumer with Kafka Producer](scenarios/workshop-5-multi-topic-producer.md) | Advanced | 60–90 min |

### Learning Paths

- **Beginner** (30–45 min): Workshop 1 only
- **Intermediate** (3–4 hours): Workshops 1 → 2 → 3
- **Advanced** (6+ hours): All workshops 1 → 5
